# 🚀 Personal Milestone Tracker

## 📌 Tech Stack
- React.js
- Node.js + Express
- Tailwind CSS

## ✨ Features
- Add milestones
- View milestones
- Form validation
- Loading & error handling

## 🛠️ Run Locally

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
